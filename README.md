# Inselberg Website v3

React/Vite rebuild of the Inselberg v2 design with shared layout components and centralized content.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Shared header and footer live in `src/components`.
- Approved page copy and solution data live in `src/data/content.js`.
- Core pages live in `src/pages`.
- Static images are served from `public/assets/images`.
